Markdown Viewer — Mark
A desktop Markdown viewer and exporter built as a standalone Windows application, capable of rendering Markdown as styled HTML or displaying raw text — switchable with a single keyboard shortcut. It supports full-featured search, print, PDF and DOCX export, dark/light theme switching, a recent files history, and a system-wide CLI launcher (Mark.exe) for opening files from anywhere on the command line. Built with PyQt6 and packaged as a native Windows executable, it eliminates the need for a browser or any online tool to read and share Markdown files.

Cambridge Syllabus Management Toolkit
A two-script Python toolkit built for Cambridge programme coordination. Script 1 automates the bulk downloading of all Cambridge IGCSE and AS & A Level syllabuses (155+ subjects) from the Cambridge International website, organising them into a clean folder structure by qualification level and subject code. Script 2 reads the first page of each downloaded syllabus PDF, extracts exam series availability (March / June / November) and administrative zone data, and generates a neatly formatted Excel report with colour-coded YES/NO values and separate sheets for IGCSE and AS & A Level. Together they eliminate hours of manual work in Cambridge curriculum coordination.
